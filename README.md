## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (optional ->) Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key if you want to have insights about the songs.
3. Run the app:
   `npm run dev`
